# Data Science & Analytics Projects
---

## Data Science & Analytics Projects

This repository contains hands-on data science and analytics projects demonstrating end-to-end problem solving using **Python, machine learning, statistical modeling, and exploratory data analysis**. The notebooks focus on translating business problems into analytical solutions with clear insights and evaluation.

---

### Project Descriptions

#### **Allocation & Covering Models**

**File:** `Allocation&CoveringModels-1.ipynb`
Implements optimization and allocation models to solve resource distribution and coverage problems. Focuses on formulating constraints, objective functions, and evaluating optimal solutions for real-world planning scenarios.

---

#### **Automated Feedback Analyzer**

**File:** `AutomatedFeedbackAnalyzer (1).ipynb`
Builds a text analytics pipeline to analyze and categorize user feedback using NLP techniques. Includes preprocessing, feature extraction, sentiment analysis, and insight generation to support product and customer experience decisions.

---

#### **Churn Analysis Lab**

**File:** `Churn_Analysis_Lab.ipynb`
Performs customer churn analysis using exploratory data analysis and machine learning models. Covers feature engineering, classification modeling, model evaluation, and interpretation of drivers influencing customer retention.

---

#### **House Pricing Practice**

**File:** `HousePricingPractice.ipynb`
Develops regression models to predict housing prices based on property features. Includes data cleaning, exploratory analysis, feature selection, model training, and performance evaluation.
